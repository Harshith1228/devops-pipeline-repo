Automating CI/CD pipeline with AWS, JENKINS AND GITHUB

About the Project

A streamlined deployment pipeline is implemented, automating the process of pulling updates from GitHub, testing changes, and deploying static website files to AWS S3. The solution ensures real-time updates, scalability, and global accessibility for hosted content. It leverages Jenkins, GitHub webhooks, and AWS services for efficient and reliable delivery.

Technologies Used

  • Amazon Web Services (AWS)
    AWS is a cloud computing platform that provides a broad range of services like compute, storage, networking, databases, and machine learning.
  • Jenkins
    Jenkins is an open-source automation server used to automate tasks related to building, testing, and deploying software.
  • GitHub repository
    Github repository is a file version keeping software that keeps track of all the versions of the file so far.
  • AWS S3
    AWS S3 is utilized in the project as the deployment and hosting platform, leveraging Amazon S3 for storing and serving the static website content globally with high scalability and availability.
